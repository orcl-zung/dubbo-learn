# dubbo-learn

个人深度学习 Apache Dubbo 的工作区（目标：读通 Dubbo 3.x 源码）。学习路径为 **会用 → 懂架构 → 攻源码**。

## 目录结构

| 路径 | 内容 |
|------|------|
| `lessons/` | 渐进式课程（HTML，含交互测验、架构图、深色模式、代码高亮/一键复制） |
| `reference/` | 术语表、架构与调用流程速查（含「概念→源码入口」对照表） |
| `assets/` | 可复用组件（排版样式、测验、架构图、代码高亮/复制、主题切换、favicon） |
| `learning-records/` | 学习记录（关键认知与踩坑教训） |
| `MISSION.md` / `RESOURCES.md` / `NOTES.md` | 学习使命、权威资源、教学备忘 |
| `dubbo-learn-practice/` | 配套可运行 demo（Spring Boot + Nacos，api/provider/consumer 三模块） |

## 从哪开始

打开 `lessons/0001-what-is-dubbo-rpc-architecture.html`（第 1 课：RPC 与 Dubbo 是什么、五个角色、一次调用流程）。

> 课程页面是本地 HTML，直接用浏览器打开即可。右上角有深色/浅色切换，代码块悬停可一键复制。
